*********************************************
VSCODEで同一単語を選択
*********************************************
Last Updated on 2021-08-15

共通の変数名や単語を一括変換したいときに便利

次の単語を同時選択
-------------------------------
* 選択したい単語を選択
* :kbd:`Ctrl` + :kbd:`D` 

全単語を同時選択
-----------------------------
:kbd:`Ctrl` + :kbd:`D` の連打も良いが一気にやる方法もある

* 選択したい単語を選択
* :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`L`
