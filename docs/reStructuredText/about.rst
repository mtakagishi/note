reStructuredTextについて
============================================

reStructuredTextの特徴
---------------------------------
* マークダウンみたいな記述法のPython版
* Python製でPythonコミュニティで長年使われている
* HTMLやPDFなどへ出力可能
* RST、reST、ReSTなどに略される
* 役割はマークダウンと同じ。 [#markdown-vs-rst]_
* Sphinxで使用。Sphinxは設定すればマークダウンも使える
* docstringでも利用
