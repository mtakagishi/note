*********************************************
VSCODEで矩形選択する方法
*********************************************
Last Updated on 2021-08-15

キーボードのみ
-----------------
* :kbd:`Ctrl`  +  :kbd:`Alt`  + :kbd:`Shift` + :kbd:`↑`/:kbd:`↓`/:kbd:`←`/:kbd:`→`

キーボードのみ（2段階)
----------------------------------
* :kbd:`Ctrl`  +  :kbd:`Alt`  + :kbd:`↑`/:kbd:`↓`

で上下方向の範囲を決めてからの

* :kbd:`Shift` + :kbd:`←`/:kbd:`→` で範囲選択

マウス使う場合
----------------------------------
* :kbd:`Alt`  +  :kbd:`Shift`  +  ドラッグ
