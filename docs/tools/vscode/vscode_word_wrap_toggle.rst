*********************************************
VSCODEでテキストの折り返しON/OFFを切り替える
*********************************************
Last Updated on 2021-08-15

* :kbd:`Alt` + :kbd:`Z`  

.. note:: 

  | VSCODEの表示画面ないでテキストの折り返しをON/OFF手軽に切り替えるショートカットです。
  | コーディングの際には、色んな窓を開けながら作業するため、
  | 意外に重宝するショートカットです。
