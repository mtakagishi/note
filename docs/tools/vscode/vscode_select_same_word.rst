VSCodeエディタ使用時に選択単語を同時選択
*********************************************
Last Updated on 2021-08-15

共通の変数名や単語を一括変換したいときに便利

選択している単語について次に出現する単語を同時選択する
--------------------------------------------------------------

* :kbd:`Ctrl` + :kbd:`D` 

選択しているについて文書内の全ての単語を同時選択
----------------------------------------------------------

* :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`L`
