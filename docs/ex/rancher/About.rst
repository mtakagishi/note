RANCHERとは
==============================
* コンテナ管理基盤、オーケストレータ基盤

RANCHERでできること
==============================
* コンテナレジストリからkubernetesをプロビジョニング
* ノードプロビジョニングしてそこにk8sを構築
* 別のkubernetesクラスタで実行しているからインポート
* 数千のkubernetesクラスタ、ノードを一括管理
